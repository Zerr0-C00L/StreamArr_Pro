# StreamArr Web UI

Modern React + TypeScript web interface for StreamArr, inspired by Radarr/Sonarr.

## Quick Start

\`\`\`bash
npm run dev  # Start at http://localhost:3000
\`\`\`

## Features
✅ Dashboard with library stats  
✅ Movies library with grid view  
✅ Search and filtering  
✅ Movie details modal  
✅ Dark theme UI  

## Tech Stack
React 18 • TypeScript • Vite • TanStack Query • Tailwind CSS • Lucide Icons

See full documentation in project root.
